# Product Overview

A personal developer portfolio website for Ibrahim Memon, a Senior Software Engineer specializing in React Native, mobile apps, UX, and JavaScript technologies.

## Purpose

Single-page portfolio site showcasing professional experience, projects, and contact information. Deployed on Vercel with analytics and speed insights enabled.

## Key Sections

- **Banner**: Hero section with photo, tagline, and contact CTA
- **About**: Background and skills overview
- **Experience**: Professional work history
- **Projects**: Showcase of completed work with screenshots
- **Footer**: Social links and closing content

## SEO & Analytics

- Open Graph metadata for social sharing
- JSON-LD structured data (Person schema)
- Google Analytics via GTM
- Vercel Analytics and Speed Insights
- Sitemap and robots.txt generation
